Database Builder

Pulls every recipe from TheMealDB (free, no account) and combines it with your current recipes-db.json into one big file for your repo.

Run this once (or whenever you want to refresh). It makes 26 small requests — one per letter of the alphabet — directly from your browser to TheMealDB, then hands you the finished file. Nothing is uploaded anywhere.

Your download of recipes-db.json has started. To put it live:

  1. In your GitHub repo, open the existing recipes-db.json.
  2. Replace its contents with the downloaded file (or delete and re-upload).
  3. Commit — GitHub Pages redeploys automatically, and every visitor gets the full collection.
